# ZORA FWD Event Quiz App

Welcome to the **ZORA FWD Event quiz app**!  
This app allows participants to register, attempt a quiz, provide feedback and see their results, while also providing a simple admin dashboard to view all participants and top winners.

---

##  Features

- **User Registration:**  
  Simple landing page collects name, email, and optional phone number.
- **Quiz Attempt:**  
  After registration, users answer 10 quiz questions.
- **Feedback Submission:**  
  After completing the quiz, users can provide feedback on their experience of zora app.
- **Score Display:**  
  At the end, users see their score.
- **Admin Dashboard:**
  - **admin_score:** View all users with their scores and feedback.
  - **admin_winner:** View the top 3 winners, determined by highest score (ties broken by earliest registration).
